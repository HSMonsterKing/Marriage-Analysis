# -
婚姻分析
# 📊 那些人較易結婚或離婚專題

本專題以公開資料為基礎，透過資料工程與資料分析流程，探討「不同族群結婚與離婚的分布與趨勢」，並透過視覺化呈現分析結果，提供具體洞察。

---

# 🧭 專題流程

本專題依循標準資料分析流程進行，包含以下階段：

---

## 1️⃣ 資料尋找（Data Discovery）

- 在多個平台中搜尋資料，例:kaggle,中華民國內政部,dataset research google...  
- 使用政府開放資料平台作為主要資料來源  
- 搜尋與結婚、離婚及初婚相關之統計資料  
- 篩選具分析價值之資料集（年齡、性別、婚姻狀況等）

---

## 2️⃣ 資料收集與下載（Data Collection）

- 使用 Python (`requests`) 自動下載資料
- 解決實務問題（如 SSL 憑證錯誤）
- 將資料儲存為 CSV 格式以供後續處理

---

## 3️⃣ 資料清理與轉換（Data Cleaning）

- 處理項目欄中的資料正規化
- 統一年齡區間（解決不同資料表區間不一致問題）
- 建立標準化欄位（如：統一年齡區間）

---

## 4️⃣ 資料存入（ETL 流程）

- Extract：從 Python / CSV 取得資料  
- Transform：資料清理、格式統一、欄位轉換  
- Load：以CSV或Microsoft Azure SQL Server匯入 Power BI 進行後續分析  

---

## 5️⃣ 資料分析（Data Analysis）

- 計算關鍵資料
  - 結婚統計
  - 離婚統計
  - 初婚統計
  - 原住民婚姻狀況統計
- 分析不同族群差異：
  - 年齡分布
  - 性別差異
  - 教育程度
  - 地區
  - 婚姻狀況

---

## 6️⃣ 深入探討（Insight Exploration）

- 找出高機會族群（結婚較高）
- 找出高風險族群（離婚較高）
- 分析不同年齡層的婚姻趨勢
- 探討潛在影響因素

---

## 7️⃣ Power BI 視覺化（Data Visualization）

使用 Power BI 建立互動式 Dashboard：

- 📊 年齡 vs 結婚率（長條圖）
- 📊 年齡 vs 離婚率（長條圖、圓餅圖）
- 📊 性別與婚姻狀態分析（長條圖）

---

## 8️⃣ 成果呈現（Presentation）

- 將分析結果整理為簡報
- 說明：
  - 分析目的
  - 方法與流程
  - 主要發現
  - 結論

---

# 🛠 使用技術

- Python（資料下載 / 自動化）
- Excel（初步資料整理）
- DAX（資料欄位連結、統計數據及美化整理）
- Power BI（資料建模與視覺化）
